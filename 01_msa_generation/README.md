# Step 1: Multiple Sequence Alignment (MSA) Generation

## Overview

The first step of the pipeline is to generate a multiple sequence alignment (MSA) for the target antigen sequence. The MSA captures evolutionary information from homologous protein sequences and serves as the primary input for the downstream fitness prediction using TranceptEVE.

In this workflow, MSAs are generated using **ColabFold**, which employs the MMseqs2 search pipeline to retrieve homologous sequences and construct the alignment.

---

## Input

A FASTA file containing the amino acid sequence of the target antigen.

Example:

```fasta
>Target_Antigen
MKTIIALSYIFCLVFADYKDDDDK...
```

---

## Tool

MSA generation is performed using the official **ColabFold AlphaFold2 notebook**:

https://colab.research.google.com/github/sokrypton/ColabFold/blob/main/AlphaFold2.ipynb

Follow the instructions provided in the notebook to upload the antigen sequence and generate the MSA.

---

## Output

The required output from this step is the generated MSA file in **A3M** format.

Example:

```text
target_antigen.a3m
```

This file will be used as the input for **Step 3: TranceptEVE Fitness Prediction**.

---

## Procedure

1. Open the ColabFold notebook.
2. Upload or paste the FASTA sequence of the target antigen.
3. Run the notebook following the default workflow.
4. After the prediction completes, download the generated **`.a3m`** MSA file.
5. Save the MSA file for use in the downstream fitness prediction step.

---

## Notes

* Ensure that the FASTA sequence corresponds to the same wild-type antigen sequence used throughout the remainder of the pipeline.
* The quality and depth of the MSA can influence the accuracy of downstream evolutionary fitness prediction.
* Only the generated **A3M MSA** is required for the subsequent steps of this workflow; protein structure prediction results generated by ColabFold are not used in this pipeline.
